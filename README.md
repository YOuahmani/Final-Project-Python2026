# Final-Project-Python2026

Offline chatbot. Once you hit play it should pop a discord styled chat box that lets you type in commands such as
if you type addmsg and type whatever you want then the bot will reply with message saved.

if you type sendall it will send all of the messages that you added and if you type clearall then it will clear all the saved messages.

A special feature is if you type ping the bot will respond with pong.

There are no bugs that I know of.

I used the concepts of JSON, Tkinter GUI, and collections.
